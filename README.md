# Portifólio Acadêmico - Nicolle Coelho

##Sobre
Esse repositório foi desenvolvido com o intuito de organizar e apresentar projetos acadêmicos, experiências, conhecimentos e trabalho realizado da disciplina Introdução a computação
O projeto utiliza GitHub e GitHub Desktop para controle de organização, documentação, funcionando como um portifólio acadêmico e profissional

##Objetivos
- Utilizar o GitHub como ferramenta de organização
- Desenvolver um portifólio acadêmico
- Publicar projetos utilizando o GitHub Pages
- Praticar organização de repositórios profissionais
